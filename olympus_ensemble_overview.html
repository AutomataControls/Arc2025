OLYMPUS Decision Process

Phase 1: Pattern Recognition Council

# All models analyze the input task pattern_votes = { 'minerva': 'composite_transformation', 'atlas': 'rotation_90_degrees', 'iris': 'color_mapping_detected', 'chronos': 'no_temporal_pattern', 'prometheus': 'standard_pattern' } # Weighted voting based on specialization detected_pattern = weighted_consensus(pattern_votes) # Result: "rotation_with_color" (Atlas + Iris lead)

Phase 2: Dual Strategy Prediction

🌐 Weighted Ensemble

All models contribute their predictions with weights based on:

  • ✓ Pattern specialization match
  • ✓ Individual model confidence
  • ✓ Historical accuracy
  • ✓ Consensus strength

Best for: Complex multi-aspect patterns

🎯 Specialist Focus

Only expert models for the detected pattern participate:

  • ✓ Rotation → ATLAS dominates
  • ✓ Color → IRIS leads
  • ✓ Sequence → CHRONOS predicts
  • ✓ Novel → PROMETHEUS creates

Best for: Clear single-pattern tasks
